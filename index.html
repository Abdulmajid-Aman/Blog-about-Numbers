JavaScript Data Types and Number Methods

Data Types

  • Numbers: Represent numeric values, e.g., 42, 3.14
  • Strings: Sequences of characters, e.g., 'Hello', "World"
  • Null: Represents an intentional absence of any object value
  • Undefined: Indicates a variable has been declared but not assigned a value
  • Objects: Collections of key-value pairs, e.g., { name: 'John', age: 30 }, [1, 2, 3]
  • Booleans: Logical values, e.g., true, false
  • BigInts: For representing integers with arbitrary precision, e.g., 1234567890123456789012345678901234567890n
  • Symbols: Unique and immutable values often used as object property keys

Math Operations

  • Addition (+): Adds two values or concatenates strings
  • Subtraction (-): Subtracts one value from another
  • Multiplication (*): Multiplies two values
  • Division (/): Divides one value by another
  • Parentheses ( () ): Used to group operations and ensure proper order of precedence

Number Methods

  • Math.round(): Rounds a number to the nearest integer
  • Math.ceil(): Rounds a number up to the nearest integer
  • Math.floor(): Rounds a number down to the nearest integer
  • Math.trunc(): Removes the decimal part of a number
  • Math.sign(): Returns -1 for negative numbers, 1 for positive numbers, and 0 for zero
  • Number.isInteger(): Checks if a value is an integer
  • Number.toString(): Converts a number to a string
  • Number.toFixed(): Formats a number to a specified number of decimal places
  • Number.valueOf(): Returns the primitive value of a number object
  • Number.toExponential(): Returns a string representing the number in exponential notation
  • Number.toPrecision(): Formats a number to a specified length

Converting Data to Numbers

  • Number(): Converts any value to a number
  • parseInt(): Parses a string and returns an integer
  • parseFloat(): Parses a string and returns a floating-point number
